# 🥊 Boxeo Piera - Web Oficial

Landing page informativa para **David Hidalgo**, entrenador personal de boxeo en el _Boxeo Piera Boxing Club_. Diseñada como un _One-Pager_ enfocado en la conversión, la accesibilidad móvil y un impacto visual fuerte.

## 🌟 Características Principales

- **Diseño "Mobile-First":** Totalmente adaptable a dispositivos móviles, tablets y escritorio.
- **Regla de Color 70/20/10:** Paleta de colores agresiva y elegante (Negro oscuro 70%, Blancos/Grises 20%, Rojo de acento 10%).
- **Animaciones CSS:** Efectos visuales de impacto como el "aura" roja difuminada y el logotipo flotante en la sección Hero.
- **Menú Hamburguesa Interactiva:** Navegación suave (Smooth Scrolling) controlada con JavaScript puro.
- **Optimización de Imágenes:** Uso de la propiedad `aspect-ratio` y `mix-blend-mode` para la integración perfecta de imágenes cuadrada.
- **Iconos SVG Integrados:** Sin dependencias externas para maximizar la velocidad de carga.

## 🛠️ Stack Tecnológico

El proyecto está construido sin frameworks para garantizar la máxima velocidad y simplicidad:

- **HTML5** (Semántico)
- **CSS3** (Variables nativas, Flexbox, Grid y animaciones personalizadas)
- **JavaScript** (Vanilla JS para manipulación del DOM)
